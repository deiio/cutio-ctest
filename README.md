# Cutio-CTest

Cutio-CTest is a unit testing framework for C

Feel free to fork, modify and improve it!

# Example

Take a look at the example file `cutio-ctst_example.c` to see
how to use this framework.

## Compile

```shell
cd cutio-ctest
cmake -B build
cmake --build build
./build/cutio-ctest_example
```
